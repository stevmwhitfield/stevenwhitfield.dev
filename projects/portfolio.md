## Project Overview

My personal website and the one you are on right now. It serves to introduce myself and show some of
my recent work.

## Technologies Used

- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ESLint](https://eslint.org/)
- [Prettier](https://prettier.io/)

## What I Learned

This is the first project where I extensively used a linter. ESLint's strict rules helped me
maintain a consistent coding style throughout the project, making the code more readable and
maintainable. The real-time linting identified errors in my code, which was especially useful for
TypeScript.

This is also the first project where I included a theme toggle. Configured using CSS variables mixed
with Tailwind, it searches for a media query or user preference from local storage and applies the
correct color palette.
