## Project Overview

The official website for Chawla Media, a small business owned by my good friend Sunisth Chawla, to
showcase his work as a photographer and videographer.

Built on Gatsby and styled with SCSS. It uses GraphQL to fetch content from Strapi CMS.

## Technologies Used

- [Gatsby](https://www.gatsbyjs.com/)
- [SCSS](https://sass-lang.com/documentation/syntax/)
- [Strapi](https://strapi.io/)
- [GraphQL](https://graphql.org/)

## What I Learned

I offered to create the site as a way to get back into web development after high school. This
project introduced me to a number of new technologies. I started by designing responsive UI design
in Figma. With the green light on design, I went to developing the website utilizing
static-site-generation with Gatsby, CSS preprocessors with SCSS, content management with Strapi, and
querying data with GraphQL.

## Future Plans

A new version is currently in the works with an improved UI and faster performance. It will use
Next.js to serve a combination of static and server-rendered pages, Tailwind CSS for styling a
responsive UI, and Sanity for content management.
